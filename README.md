# Student Analysis Dashboard (Power BI)

## Project Overview
This project presents a Student Analysis Dashboard created using Power BI. 
The dashboard helps analyze student data such as performance, attendance, and other academic insights.

## Tools Used
- Power BI
- Excel (for dataset)
- Data Visualization

## Dataset
The dataset contains information related to students such as:
- Student Name / ID
- Marks / Performance
- Attendance
- Subject-wise scores

## Dashboard Features
- Overall Student Performance
- Subject-wise Marks Analysis
- Attendance Insights
- Comparison of Student Scores

## Key Insights
- Identify top performing students
- Analyze subject-wise strengths and weaknesses
- Track attendance patterns
- Visual representation of student performance

- Student Analysis Dashboard (.pbix)
- Dataset (.xlsx / .csv)
- Dashboard Screenshot
